## FyzVim

# Befeore Start

You MUST have one of "Nerd" fonts and ALREADY installed NeoVim

# Installing

I`ll give you only one command. Just input it in ur terminal and relax ;)

```
rm -rf ~/.config/nvim && git clone --depth=1 https://gitlab.com/configslfyz/fyzvim.git ~/.config/nvim && rm -rf ~/.config/nvim/.git
```

!!!ATTENTION!!!

It will remove your previous config
