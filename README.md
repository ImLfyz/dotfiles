# Dotfiles
My setup

## Get started
1. GNU Stow requred:
```
// Arch
sudo pacman -S stow

// Debian
sudo apt install stow

// Fedora
sudo dnf install stow
```

2. Clone:
```
git clone https://github.com/ImLfyz/dotfiles
```

3. Use that you want:
  - ["Waybar"](https://github.com/ImLfyz/dotfiles/tree/main/waybar)
  - ["Nvim"](https://github.com/ImLfyz/dotfiles/tree/main/nvim)
